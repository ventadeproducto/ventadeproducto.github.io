1.HTML5 es un estandar que sirve como referencia del software que conecta con la elaboracion de paginas web en sus difereferentes versiones, define una estructura basica y un codigo (denominado HTML) para la definicion de contenido de una pagia web, como texto, videos, juegos, entre otros.

2.CCSS3 es un leguaje de diseño grafico que permite definir y crear la presentacion de un documento estructurado escrito en un lenjuaje de marcado. Es muy usado para establecer el diseño visual de los documentos web e interfaces de usuarios escritas en HTML.

3.JavaScrip es un lenjuaje de programacion interpretado, dialecto del estandar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, debilmenete tipado y dinamico.
